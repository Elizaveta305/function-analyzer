# Отчет по проекту: "Анализатор математических функций"

## 1. Введение
### 1.1. Актуальность темы
В современном образовании важны наглядность и интерактивность. 
Анализатор функций помогает визуализировать математические понятия.

### 1.2. Цель проекта
Создать веб-приложение для автоматического анализа свойств 
математических функций и построения их графиков.

### 1.3. Задачи проекта
1. Изучить теорию математических функций
2. Освоить веб-технологии (HTML, CSS, JavaScript)
3. Изучить библиотеки Math.js и Plotly.js
4. Разработать алгоритмы анализа функций
5. Реализовать пользовательский интерфейс
6. Протестировать и отладить приложение

### 1.4. Гипотеза
Использование современных веб-технологий позволяет создать 
доступный и эффективный инструмент для анализа функций.
