# Отчет по проекту: "Анализатор математических функций"

## 1. Введение
### 1.1. Актуальность темы
В современном образовании важны наглядность и интерактивность. 
Анализатор функций помогает визуализировать математические понятия.

### 1.2. Цель проекта
Создать веб-приложение для автоматического анализа свойств 
математических функций и построения их графиков.

### 1.3. Задачи проекта
1. Изучить теорию математических функций
2. Освоить веб-технологии (HTML, CSS, JavaScript)
3. Изучить библиотеки Math.js и Plotly.js
4. Разработать алгоритмы анализа функций
5. Реализовать пользовательский интерфейс
6. Протестировать и отладить приложение

### 1.4. Гипотеза
Использование современных веб-технологий позволяет создать 
доступный и эффективный инструмент для анализа функций.

## 4. Этапы разработки
### 4.1. Структура проекта
Проект имеет модульную структуру:
- **HTML (index.html)** — пользовательский интерфейс
- **CSS (style.css)** — оформление и стилизация
- **JavaScript (script.js)** — логика работы, разделенная на модули:
  - Главный модуль (инициализация и обработка событий)
  - Модуль анализа функций (определение типа, свойств)
  - Модуль графиков (построение и визуализация)
  - Вспомогательные функции

### 4.2. Реализованные возможности
1. **Анализ функций**: 
   - Определение типа функции (квадратичная, кубическая, линейная, тригонометрическая)
   - Расчет основных свойств (область определения, нули функции)
   - Визуализация результатов в виде карточек

2. **Визуализация**:
   - Построение графиков с использованием библиотеки Plotly.js
   - Настройка осей, сетки, легенды
   - Автоматическое масштабирование

3. **Пользовательский интерфейс**:
   - Современный дизайн с использованием CSS Flexbox
   - Адаптивная верстка
   - Интерактивные элементы с обратной связью

### 4.3. Используемые технологии
- HTML5, CSS3, JavaScript (ES6+)
- Библиотеки: Plotly.js (графики), Math.js (вычисления)
- Система контроля версий: Git, хостинг: GitHub Pages
- Инструменты разработки: VS Code с Live Server
